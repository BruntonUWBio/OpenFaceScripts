VidCropper.crop_and_resize(vid, width, height, x_min, y_min, directory, resize_factor)¶

Crops a video and then resizes it

Parameters:
  • vid – Video to crop
  • width – Width of crop
  • height – Height of crop
  • x_min – x-coordinate of top-left corner
  • y_min – y-coordinate of top-left corner
  • directory – Directory to output files to
  • resize_factor – Factor by which to resize the cropped video
