CropAndOpenFace¶

runners.CropAndOpenFace.run_open_face(im_dir, vid_mode=False, remove_intermediates=False) → str¶

Runs OpenFace

Parameters:
  • im_dir – Location of images if not in video mode, location of video if in video mode
  • vid_mode – Whether or not to be in video mode (alternative is to run on an image sequence)
  • remove_intermediates – Whether or not to remove intermediate files
Returns:

Name of output video produced by OpenFace (with landmarks)
